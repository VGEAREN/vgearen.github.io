## Quick Start

**Install Hexo**

``` bash
$ npm install hexo-cli -g
```

**Setup your project**

``` bash
$ hexo init <folder>
$ cd <folder>
$ npm install
```

**Clean && Generate static files && Start the server**

``` bash
$ hexo cl && hexo g && hexo s
```

**Create a new post**

``` bash
$ hexo new "Hello Hexo"
```
